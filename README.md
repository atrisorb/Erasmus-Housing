# Erasmus-Housing
A helpful website for Erasmus students to find the perfect place to live in their host country.

## Project Description

ErasmusHousing is a web platform designed to facilitate the search for accommodation for Erasmus students (and other international/local students) within the European Union. The goal is to connect homeowners, student residence managers, and student houses with students seeking accommodation.

This repository contains the source code for the frontend (HTML, Tailwind CSS-generated CSS, JS) and backend (Python/Flask) of the website.

## Current Status

The project includes:
* Static HTML pages for the frontend with a modern and responsive design (Tailwind CSS).
* Python/Flask backend with:
    * API for managing listings (mock data).
    * User authentication system:
        * Email and Password Registration/Login (with email verification).
        * Social Login with Google.
        * Social Login with Facebook.
    * Basic messaging system between users.

**Implemented Pages:**

* `index.html`: Main page.
* `listings.html`: Dynamic listings page.
* `listing-detail.html`: Dynamic listing detail page.
* `add-listing.html`: Add listing page (requires login).
* `login.html`: Login/registration page with social and email options.
* `messages.html`: User messaging page.

## Technologies Used

* **Frontend:** HTML5, Tailwind CSS (compiled via `@tailwindcss/cli`), JavaScript.
* **Backend:** Python 3, Flask, Flask-Login, Flask-Mail, Flask-CORS, google-auth-oauthlib, requests, Gunicorn.
* **Frontend Build:** Node.js, npm, `@tailwindcss/cli`.

## Getting Started (Local Development)

1.  **Clone the repository.**
2.  **Set up the Python Environment:**
    * Create a Python virtual environment and activate it:
        ```bash
        python -m venv venv
        source venv/bin/activate  # On Windows: venv\Scripts\activate
        ```
       
    * Install Python dependencies:
        ```bash
        pip install -r requirements.txt
        ```
       
3.  **Set up the Node.js and Tailwind CSS Environment:**
    * Ensure you have Node.js and npm installed.
    * In the project's root directory (`ErasmusHousing`), install Node.js dependencies:
        ```bash
        npm install
        ```
    * Start the Tailwind CSS build process in "watch" mode (it stays active and recompiles CSS when you make changes):
        ```bash
        npm run watch:css
        ```
        (This command assumes you have a `"watch:css"` script defined in your `package.json`, such as: `"watch:css": "npx @tailwindcss/cli -i ./src/input.css -o ./static/css/output.css --watch"`)

4.  **Configure Environment Variables for Flask:**
    * Create a `.env` file in the project root (this file is in `.gitignore` and will not be uploaded to GitHub).
    * Add the necessary environment variables to the `.env` file as specified in the original README section (FLASK_SECRET_KEY, GOOGLE_CLIENT_ID, etc.).
    * **Important:** Ensure the redirect URIs for Google and Facebook in their respective developer consoles are set correctly (e.g., `http://127.0.0.1:10000/login/google/callback` for local development, or the corresponding URL if using a different port).

5.  **Start the Flask server:**
    * Open a **new terminal** (leave the one with `npm run watch:css` running).
    * Activate the Python virtual environment if necessary.
    * Run:
        ```bash
        python app.py
        ```
    The backend will be running (by default on `http://127.0.0.1:10000` or your configured port).

6.  **Open the site in your browser:**
    * Navigate to `http://127.0.0.1:10000/index.html` (or the port you use) to see the site. The HTML pages will now use the `output.css` file generated by Tailwind and served by Flask.

## Production Build

* To generate the optimized CSS file for production, you can use a script similar to this in your `package.json`:
  ```json
  "scripts": {
    "build:css": "npx @tailwindcss/cli -i ./src/input.css -o ./static/css/output.css --minify"
  }

## Come Iniziare (Sviluppo Locale)

1.  **Clona la repository.**
2.  **Configura l'Ambiente Python:**
    * Crea un ambiente virtuale Python e attivalo:
        ```bash
        python -m venv venv
        source venv/bin/activate  # Su Windows: venv\Scripts\activate
        ```
       
    * Installa le dipendenze Python:
        ```bash
        pip install -r requirements.txt
        ```
       
3.  **Configura l'Ambiente Node.js e Tailwind CSS:**
    * Assicurati di avere Node.js e npm installati.
    * Nella cartella principale del progetto (`ErasmusHousing`), installa le dipendenze Node.js:
        ```bash
        npm install
        ```
    * Avvia il processo di build di Tailwind CSS in modalità "watch" (rimane attivo e ricompila il CSS quando fai modifiche):
        ```bash
        npm run watch:css 
        ```
        (Questo comando presuppone che tu abbia uno script `"watch:css"` definito nel tuo `package.json`, come ad esempio: `"watch:css": "npx @tailwindcss/cli -i ./src/input.css -o ./static/css/output.css --watch"`)

4.  **Configura le Variabili d'Ambiente per Flask:**
    * Crea un file `.env` nella root del progetto (questo file è in `.gitignore` e non verrà caricato su GitHub).
    * Aggiungi le variabili d'ambiente necessarie al file `.env` come specificato nella sezione originale del README (FLASK_SECRET_KEY, GOOGLE_CLIENT_ID, ecc.).
    * **Importante:** Assicurati che gli URI di redirect per Google e Facebook nelle rispettive console developer puntino correttamente (es. `http://127.0.0.1:10000/login/google/callback` per lo sviluppo locale, o l'URL corrispondente se usi una porta diversa).

5.  **Avvia il server Flask:**
    * Apri un **nuovo terminale** (lascia quello con `npm run watch:css` in esecuzione).
    * Attiva l'ambiente virtuale Python se necessario.
    * Esegui:
        ```bash
        python app.py
        ```
    Il backend sarà in esecuzione (di default su `http://127.0.0.1:10000` o la porta che hai configurato).

6.  **Apri il sito nel browser:**
    * Naviga a `http://127.0.0.1:10000/index.html` (o la porta che usi) per vedere il sito. Le pagine HTML ora useranno il file `output.css` generato da Tailwind e servito da Flask.
